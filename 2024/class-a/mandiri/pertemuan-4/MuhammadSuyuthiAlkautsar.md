# Apakah Perbedaan Ideologi Antara Open Source dan Closed Source Memengaruhi Kebijakan teknologi yang diterapkan Negara atau Organisasi
## Latar Belakang
 Perangkat lunak dapat dibagi menjadi open source dan closed source. Open source dapat dipublikasikan secara luas dan dikerjakan oleh banyak orang sekaligus contohnya seperti pengembangan software linux, hal ini dapat meningkatkan produktivitas tetapi juga memiliki banyak kekurangan jika tidak di atur dengan terkoordinasi. Sedangkan closed source menyimpan kodenya secara privat sehingga hanya perusahaan atau pembuat yang bisa melihatnya. Contohnya microsoft dan apple, hal ini membuat ketergantungan update dari developer.
